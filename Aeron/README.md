# Aeron

`Aeron` is the root of the fantasy world.

## Structure
- `mythopedia/` holds organized reference material.
- `primal_language/` holds foundational words and concepts, from the highest abstractions to the smallest details.
- `sagas/` holds story-shaped tellings of the world.

## Conventions
- Entries under `Aeron` are Markdown files by default.
- `mythopedia` is canon-first and reference-oriented.
- `sagas` is prose-first and story-oriented.
- `primal_language` grows as a hierarchy of concept files when new foundational terms appear.
