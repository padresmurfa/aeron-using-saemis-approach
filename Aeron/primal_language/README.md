# Primal Language

`primal_language` is the home of first-order words and concepts that shape creation.

## Use
- Add Markdown files for foundational concepts introduced by the sagas.
- Organize concepts into folders only when a clear hierarchy emerges.
- Start from the highest abstractions and move downward into narrower expressions.

## Current Structure
- `high_abstractions/` for first-order metaphysical roots
- `threshold_events/` for foundational transitions that alter the shape of reality
