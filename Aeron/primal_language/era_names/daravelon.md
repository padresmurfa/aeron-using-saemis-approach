# Daravelon

- Rendered form: `Daravelon`
- Gloss: Reionization Era; the age of first stellar ignition and outward-borne created radiance
- Tier: era name

## Meaning

`Daravelon` names the age in which self-held light no longer remains only inwardly preserved, but is kindled into sustained outward radiance. In this era, the first stable luminous centers appear, darkened space grows answerable again, and created beings begin to illuminate regions beyond themselves.

## Scope

`Daravelon` carries:

- First stellar ignition
- Sustained interior-to-exterior radiance
- Created illumination across distance
- Luminous stewardship inside creation
- Domain-shaping fire before soul-bearing ethics

## Canonical Status

`Daravelon` is the primal-language name for the Reionization Era.
