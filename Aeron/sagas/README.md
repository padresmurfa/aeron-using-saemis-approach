# Sagas

`sagas` is the narrative layer of Aeron.

## Use
- Write events, beings, and epochs in story form.
- Preserve voice and dramatic shape without breaking canon.
- Treat `mythopedia/` and `primal_language/` as truth sources for consistency.

## Current Structure
- `tales_of_creation/` for origin-cycle sagas and creation ages
- `celestials/` for luminaries, moons, and constellation sagas
