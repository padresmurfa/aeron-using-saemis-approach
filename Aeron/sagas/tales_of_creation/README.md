# Tales Of Creation

`tales_of_creation` holds the saga cycle that describes the making of the world.

## Use
- Keep stories here focused on origins, first acts, first beings, and the ordering of creation.
- Preserve timeline coherence across all entries in this folder.
- Surface any new foundational concepts for extraction into `Aeron/primal_language/`.
