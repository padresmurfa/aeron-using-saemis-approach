# Celestials

`celestials` holds saga-form tellings for Aeron's luminaries, moons, and named constellations.

## Use
- Keep these tales centered on the sky as a living theater of meaning rather than as technical description alone.
- Preserve alignment with `Aeron/mythopedia/astrology/` and `Aeron/primal_language/`.
- Introduce canonized primal headwords and English glosses together when the tale first depends on them.
