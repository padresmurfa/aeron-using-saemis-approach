# Timeline

`timeline` indexes Aeron's canon chronology without rewriting the source material.

## Use
- Record ordered milestones with clear temporal relevance.
- Keep each milestone in one canonical era only.
- Prefer links back to `mythopedia/`, `sagas/`, and `primal_language/` instead of long retellings.
- Note canon gaps explicitly instead of inventing missing events.
- Use canonical era roots and English glosses together when an era has a primal-language headword, and avoid English-only concept references once a primal headword exists except in explicit glossing contexts.
