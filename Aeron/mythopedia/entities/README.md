# Entities

This directory contains canonical entries for beings and named intelligences in Aeron. As of the Planck Layer update, the canonical names are:

- Aru (formerly creator.md)
- Loran (formerly c.md)

See each file for details. All future entity files must use the canonical primal language name, with English glosses used only as gloss. After first introduction, prefer the primal name alone or `PrimalWord (English gloss)` rather than the English gloss by itself.
