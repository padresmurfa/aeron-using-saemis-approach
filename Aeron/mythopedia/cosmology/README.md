# Cosmology

This directory contains canonical reference entries for the fundamental metaphysical structure of Aeron. Each file covers a distinct layer or law of reality, from the pre-Big-Bang state through the emergence of structure, oscillation, and the Primordials.

## Key Entries
- `pre_big_bang_state.md`: The absolute condition before all structure.
- `divine_oscillation.md`: The first internal relational structure.
- `primordial_unfolding.md`: The emergence of the Primordials as structural grammar.
- `planck_layer_primordials.md`: The 25 irreducible axioms of existence (Planck Layer).

Refer to these files for the metaphysical and ontological rules that govern all subsequent mythic events and entities.
