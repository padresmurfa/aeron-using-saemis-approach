# The First Pulse

## Status

The First Pulse is the boundary event that ends the pre-Big-Bang state and begins creation proper. In the current canon layer, it is established in principle but not yet fully narrated in detail.

Primary references:
- [Pre-Big-Bang State](../cosmology/pre_big_bang_state.md)
- [The Creator](../entities/creator.md)
- [The Stillness Before the First Pulse](../../sagas/tales_of_creation/the_stillness_before_the_first_pulse.md)

## Definition

The First Pulse is:

- The Creator's first act of self-differentiation
- The beginning of divine oscillation
- The threshold through which relation becomes possible
- The first opening of sequence and causation

## What The First Pulse Is Not

The First Pulse is not:

- An explosion into empty space
- A reaction to loneliness or lack
- A correction to deficiency
- An event caused by something outside the Creator

There is no external medium into which the First Pulse is launched. It arises wholly from the Creator's own self-expression.

## Consequences

After the First Pulse, the following become possible:

- Distinction
- Relation
- Sequence
- Causation
- Resonance
- Harmony

All later cosmic and metaphysical orders stand downstream of this act.

## Canon Boundary

This entry establishes only the nature of the First Pulse as the first act of self-differentiation. It does not yet define the full sequence by which the Primordials or the material cosmos emerge.
