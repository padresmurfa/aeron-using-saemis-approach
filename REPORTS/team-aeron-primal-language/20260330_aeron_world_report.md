# Primal Language Team Report

PRIMAL LANGUAGE TEAM REPORT
===========================

Scope: canonize `Aeron` as the primal proper name of the world

Findings:
- The current corpus named the setting `Aeron` throughout narrative use, but the primal-language lexicon did not yet define `Aeron` as a canonical headword.
- This made the world's name visible in prose without giving it lexical status equal to `Solaryth`, `Nyxorys`, or the named `Seravel`.
- The cleanest repair is to preserve `Aeron` as a direct primal planetary headword rather than forcing it through an artificial retro-derivation from an existing family.

Lexical Decision:
- `Aeron`: the canonical proper name of the central world

Grammar Note:
- `Aeron` is treated as a preserved proper form whose authority comes from canonical usage, not from transparent decomposition into one dominant tracked family.

Dictionary Work:
- Added bidirectional dictionary support for `Aeron`.
- Added a structural-principle entry clarifying that `Aeron` names this world specifically and does not replace the generic category of all worlds.

Verdict: Accept. `Aeron` is now canonized as a primal headword for the world itself.
